# base_csv_generator
出力するCSVファイルのヘッダーとcsv作成用の雛形の自動作成を行う

## 前提
pythonがインストールされている
pandas(ライブラリ)がインストールされている
サンプル用のcsvデータがある

## 使い方
サンプル用のcsvデータをダウンロードしてこのフォルダーの直下に置く
実行するときは以下のコマンドを叩く
```
python main.py csvのファイル名
```

すると以下ように動作する  
ヘッダー　headers.txtにかきこまれる  
csv作成用の雛形　colmuns(今度名前変えます)

実行するたびにheaders.txtとcolmuns.txtは上書きされます
